Tomcat管理功能使用建议：
1.Windows下启动会存在控制台弹窗和cmd命令弹出，控制台弹窗关闭则Tomcat关闭，建议修改Tomcat的bin目录下：
【setclasspath.bat】文件末尾81行左右的 【set _RUNJAVA="%JRE_HOME%\bin\java.exe"】
==> 【set _RUNJAVA="%JRE_HOME%\bin\javaw.exe"】 则后台运行不弹出控制台弹窗，cmd弹窗可关闭
2.Tomcat 管理功能还不健全，请生产环境谨慎使用，需测试环境多加测试
3.Tomcat新增是指 配置服务器已有Tomcat信息，配置里面的 端口号实际是Tomcat的server.xml的端口配置
[[${item.name}]]
